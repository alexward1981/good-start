# Good Start

This is a standard nuxt installation with storybook, apollo-link, element-ui, vuex, vue-router, standardjs and a few basic config files pre-installed and set up with a great architecture.

## Getting started

To get started, simply clone the repository on to your machine and install the dependencies.

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate

# run tests
$ npm run test

# run tests in watch mode
$ npm run test:watch

# regenerate test snapshots
$ npm run test -u
```
